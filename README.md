Web-based application for building questionnaires. Entry task for ET-school.

What is currently implemented:
1. Main Page: loads all avaliable quizes from database. Has a button to create a new quiz. Each quiz has a button to complete it.
2. Quiz creation page: you can create a ne quiz, choose what counts as a correct answer, and send this quiz to database (later is shown on main page).
3. Quiz page: you can beat any quiz. After completion, it shows your results.
